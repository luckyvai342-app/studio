# **App Name**: FreeFire Apex

## Core Features:

- Secure Phone OTP Authentication: Users can register and log in securely using their phone number verified via One-Time Password (OTP), and create a basic player profile (e.g., Free Fire ID, username).
- Server-Controlled Wallet & Deposits: Users can view their wallet balance and initiate requests to add funds. All wallet balance updates, additions, and deductions are handled exclusively by secure server-side logic (e.g., Firebase Functions) to prevent client-side manipulation.
- Tournament Browsing & Secure Joining: Users can browse a list of active and upcoming Free Fire tournaments, view detailed information including entry fees and prize pools, and securely join tournaments. The entry fee deduction is fully server-controlled.
- Match & Results Management: Allows tracking the status of joined matches. Admins can efficiently report match results, verify participant placements (e.g., top 20 players), and determine winners based on verified data.
- Server-Controlled Prize Distribution & Withdrawal: Prize money is automatically and securely distributed to winners' wallets via server-side processes. Users can request withdrawals from their wallet, subject to manual approval by administrators.
- Detailed Transaction History: Users have access to a comprehensive and personalized transaction log, displaying all financial activities such as deposits, tournament entry fees, prize winnings, and withdrawal requests with their current status.
- AI-Powered Match Recap Tool: After a tournament concludes, an AI tool automatically generates an engaging textual recap highlighting key statistics, top performances, and significant moments based on collected match data.

## Style Guidelines:

- Primary color: A vibrant, energetic purple (#B34DFF) for a bold and tech-forward feel, associated with competitive spirit.
- Background color: A deep, muted plum (#231E24) providing a dark and immersive canvas that highlights the primary and accent colors.
- Accent color: A bright, electric blue (#6699FF) used for calls to action, highlights, and critical information to ensure high contrast and draw user attention.
- Headlines and prominent text will use 'Space Grotesk' (sans-serif) for a modern, slightly futuristic, and technical aesthetic. Body text and longer descriptions will use 'Inter' (sans-serif) for excellent readability across various screen sizes.
- Clean, sharp, and geometric vector icons that resonate with esports themes, incorporating symbols like trophies, crosshairs, and digital interfaces.
- A dark-themed, highly organized layout with clear separation of content blocks. Emphasizes tournament listings, user statistics, and intuitive navigation. Designed for a mobile-first user experience.
- Subtle yet dynamic UI animations, such as smooth transitions between views, animated loading states, and celebratory effects for winning tournaments or earning prizes, enhancing the engaging user experience.
